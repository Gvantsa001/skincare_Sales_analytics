პროექტის აღწერა

წინამდებარე პროექტი წარმოადგენს ბიზნეს მონაცემთა ანალიტიკური გადაწყვეტილებების მხარდაჭერის სისტემას, რომელიც შექმნილია კანის მოვლის (Skincare) პროდუქტების ელექტრონული კომერციის კომპანიისთვის.

სისტემის მიზანია ისტორიული გაყიდვების მონაცემების დამუშავება, მომხმარებელთა ქცევის ანალიზი, პროდუქციის ეფექტურობის შეფასება, სეზონური ტენდენციების გამოვლენა და მომავალი გაყიდვების პროგნოზირება.

პროექტი აერთიანებს მონაცემთა მეცნიერების (Data Science), ბიზნეს ინტელექტის (Business Intelligence) და მონაცემთა ანალიტიკის თანამედროვე ინსტრუმენტებს ერთიან ანალიტიკურ პლატფორმაში.

კორელაციური ანალიზის ფარგლებში შესწავლილ იქნა:

ფასდაკლება და გაყიდვები;
შეკვეთების რაოდენობა და შემოსავალი;
პროდუქტის ფასი და გაყიდული რაოდენობა;
რეიტინგი და შემოსავალი;
რეიტინგი და დაბრუნების მაჩვენებელი;
მომხმარებლის აქტივობა და Customer Lifetime Value.

პროექტში გამოყენებულ იქნა სამი პროგნოზირების მოდელი:
Seasonal Naive Forecasting- საბაზისო პროგნოზირების მოდელი, რომელიც ეფუძნება სეზონურობის პრინციპს.
ARIMA (AutoRegressive Integrated Moving Average) - მოდელი დროითი მწკრივების პროგნოზირებისთვის.
Holt-Winters Exponential Smoothing - ექსპონენციალური გასწორების მოდელი, რომელიც ითვალისწინებს ტრენდსა და სეზონურობას.
